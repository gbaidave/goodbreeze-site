## Key Learnings

### Gradient Rendering Issues
- **Problem**: Tailwind opacity classes (`/70`, `/90`) weren't rendering consistently
- **Solution**: Use explicit hex color values instead: `bg-gradient-to-br from-[#3b82f6] via-[#a855f7] to-[#00adb5]`
- **Lesson**: For critical visual elements, prefer explicit colors over opacity modifiers

### Logo Sourcing
- **Don't**: Create custom icon representations (users expect actual brand logos)
- **Do**: Use SimpleIcons.org for official brand logos
- **Pattern**: `https://cdn.simpleicons.org/[slug]/[color]` or download SVG files
- **Fallback**: For logos not in SimpleIcons, create recognizable custom designs

### Canva MCP Setup
- **Package**: `@canva/cli@latest` with `mcp` argument
- **Auth**: Requires user authentication on first use
- **Trigger**: Any Canva-related prompt after restart
- **Note**: MCP servers only load from primary workspace `.mcp.json`
